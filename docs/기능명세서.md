# 경성대학교 글로컬 컬쳐 허브 — Node.js Express 백엔드 구조 설계

## 프로젝트 개요

**KSU Culture Hub**는 경성대학교와 글로컬(Glocal) 커뮤니티를 잇는 문화 교류 플랫폼입니다.
학생·교직원 전용 혜택, 문화 이벤트, 커뮤니티 게시판, 글로벌 파트너 연계 등의 기능을 제공합니다.

---

## 기술 스택

| 항목 | 기술 |
|---|---|
| Runtime | Node.js (v20 LTS) |
| Framework | Express.js v4 |
| DB | MongoDB + Mongoose (ODM) |
| 인증 | JWT + bcryptjs |
| 파일 업로드 | Multer |
| 환경 변수 | dotenv |
| API 문서 | Swagger (swagger-ui-express) |
| 유효성 검사 | express-validator |
| CORS | cors |
| 로깅 | morgan |
| 개발 서버 | nodemon |

## 주요 API 엔드포인트

### Auth `/api/auth`
| Method | Path | 설명 |
|---|---|---|
| POST | `/register` | 회원가입 (학교 이메일 검증) |
| POST | `/login` | 로그인 → JWT 발급 |
| POST | `/logout` | 로그아웃 |
| POST | `/refresh` | 토큰 갱신 |

### Users `/api/users`
| Method | Path | 설명 |
|---|---|---|
| GET | `/me` | 내 프로필 조회 |
| PUT | `/me` | 프로필 수정 |
| GET | `/` | 사용자 목록 (Admin) |

### Events `/api/events`
| Method | Path | 설명 |
|---|---|---|
| GET | `/` | 이벤트 목록 |
| GET | `/:id` | 이벤트 상세 |
| POST | `/` | 이벤트 생성 (Admin) |
| PUT | `/:id` | 이벤트 수정 (Admin) |
| DELETE | `/:id` | 이벤트 삭제 (Admin) |
| POST | `/:id/apply` | 이벤트 신청 |

### Benefits `/api/benefits`
| Method | Path | 설명 |
|---|---|---|
| GET | `/` | 혜택 목록 |
| GET | `/:id` | 혜택 상세 |
| POST | `/:id/claim` | 혜택 수령 |

### Posts `/api/posts`
| Method | Path | 설명 |
|---|---|---|
| GET | `/` | 게시글 목록 |
| POST | `/` | 게시글 작성 |
| PUT | `/:id` | 게시글 수정 |
| DELETE | `/:id` | 게시글 삭제 |

### Partners `/api/partners`
| Method | Path | 설명 |
|---|---|---|
| GET | `/` | 파트너 목록 |
| POST | `/` | 파트너 등록 (Admin) |

## 사용자 역할 (Role)

- `student` — 경성대학교 재학생
- `staff` — 교직원
- `admin` — 플랫폼 관리자

학교 이메일 도메인 (`@ks.ac.kr`) 기반 자동 검증 적용.

---

## 검증 계획

1. `npm run dev` 로 서버 정상 기동 확인
2. Swagger UI (`/api-docs`) 에서 각 엔드포인트 확인
3. 헬스체크 엔드포인트 `GET /api/health` 200 응답 확인
