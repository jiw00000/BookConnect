README 소개
📚 BookConnect: 전공 e서적 학기 구독 플랫폼
대학생의 전공 서적 구매 부담을 완화하고 효율적인 디지털 도서 공유 생태계를 구축하는
웹 기반 학기 단위 구독 플랫폼입니다.
❌
기존 문제 (AS-IS)
전공 서적 권당 평균 40,000원 이상 고가
도서관 대출 경쟁 심화 및 기간 제한
무거운 실물 도서 지참 불편
✅
서비스 해결책 (TO-BE)
학기 단위 구독 모델로 비용 절감
수량 제한 없는 디지털 E북 환경
멀티 디바이스 N-Screen 지원
